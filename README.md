
# FitSmart

Fitsmart es na aplicación orientada al entrenamiento personalizado y al seguimiento físico del usuario.
El sistema permite registrar usuarios, almadcenar la informacion física del mismo y generar rutinas adaptadas al nivel y al objetivo que tenga dicho usuario.


## Modo de uso

El cliente debe tener la aplicación descargada e instalada.
Lo primero que verá el clientes es la pantalla para loguearse, en caso de no tener cuenta procederá a pulsar el botón de "Registro" que lo llevará a una nueva ventana. En esta ventana debe introducir los datos solicitados (Usuario, Email y Contraseña). Una vez esté registrado podrá iniciar sesión en la aplicación. Lo siguiente que debe hacer es completar el perfil físico donde debe incluir datos como la edad, el peso, la altura, el nivel físico actual y el objetivo que desea con el entrenamiento. Con el perfil ya completo, el sistema procederá a generar una rutina personalizada para dicho cliente. Ahora el usuario tendrá acceso a su rutina y a una nueva ventana donde podrá guardar su avance en forma de datos.


## Características

- Permite registro e inicios de sesión
- Registra perfil físico del usuario.
- Genera rutinas de entrenamiento personalizados en base a los datos del cliente.
- Se puede solicitar otra rutina nueva si se ha cambiado de objetivo.
- Permite llevar un registro de los entrenamientos.



## Tecnologías

**Frontend:** AndroidStudio, Java y XML

**Base de datos:** SQLite

**Control de versiones:** GitHub
